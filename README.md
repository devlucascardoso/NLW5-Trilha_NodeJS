### Downloading and Installing

**Cloning the Repository**

```
$ git clone https://github.com/devlucascardoso/NLW5-Trilha_NodeJS && cd NLW5-Trilha_NodeJS
```

**Installing dependencies**

```
$ yarn
```

_or_

```
$ npm install
```
